Excel

The source data contains more than 4000 rows of data from a survey of Kickstarter projects. The columns of the dataset are: Toggle Columns

  • id - int
  • name - Name of project - str
  • blurb - Description - str
  • goal - Value needed to succeed - int
  • pledged - Money raised - int
  • state - str
  • country - str
  • currency - str
  • deadline - Unix time - int
  • launched_at - Unix time - int
  • staff_pick - boolean
  • backers_count - Number of people supporting - int
  • spotlight - boolean
  • Category and Sub-Category - str
  • Percent Funded - str
  • Average Donation - str
  • Category - str
  • Sub-Category - str
  • Date Created Conversion - datetime
  • Date Ended Conversion - datetime
Other sheets in the file utililze pivot charts to compare the columns to each other. Charts are then generated from the pivot tables themselves! One of the sheets dives deeper into the analysis and finds the Mean, Median, Mode, Variance, and Standard Deviation for Succesfull and Failed Projects.

The source data contains 753,000 rows of data about different stock data. This is approaching the limit of a workable excel sheet. The columns of the dataset are: Toggle Columns

  • <ticker> - Stock ticker abbreviation - str
  • <date> - Date of the values taken (YYYMMDD) - datetime
  • <open> - Opening value for that day - double
  • <high> - Highest value for that day - double
  • <low> - Lowest value for that day - double
  • <close> - Closing value for that day - int
  • <vol> - Total volume of stocks for that day - str
  • Ticker - Stock ticker abbreviation - str
  • Yearly Change - Initial open value minus final close value - double
  • Percentage Changed - Initial open value minus final close value percentage - str
  • Total Stock Volume - Total stocks traded - int
There are 3 sheets in the workbook that take a look at three different years (2014, 2015, 2016) of stock data. By running the VBA macro once, ALL of the sheets are analyzed and find the Greatest Percentage Increased, Greatest Percantage Decreased, and Greatest Total Stock Volume. Futhermore, the script colors in the Yearly Change column depending on its positive or negative value.
